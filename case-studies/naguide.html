Nelson-Atkins Museum of Art: Mobile Guide

The Nelson-Atkins uses TAP to drive a mobile web app with some exciting enhancements to the standard TAP web app. There is a puzzle game that jumbles a work of art and the user must slide the tiles back into place. A Wheres Waldo style game where users are asked to find hidden items in paintings. A simulated daguerotype photo booth and encouraging photo taking with a selfie stop. All of these enhancements along with content for families combine to create a unique TAP experience.
